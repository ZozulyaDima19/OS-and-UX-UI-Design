## Хід роботи

### 1. Робота в графічному режимі Linux (GNOME)

Було розглянуто графічну оболонку GNOME.

Основні компоненти робочого середовища:

- Основне меню (Activities) — доступ до всіх програм.
- Панель швидкого доступу (Dock) — швидкий запуск додатків.
- Пошук — знаходження програм та файлів.
- Робочі столи (Workspaces) — організація відкритих вікон.
- Системна панель — індикатори мережі, гучності, живлення.

---

### 2. Запуск програм у Linux

Запуск програм можливий різними способами:

Через панель швидкого запуску — натискання на іконку в Dock.  
Через пошук у меню — введення назви програми в Activities.  
Через ярлик — подвійний клік по значку програми.

---

### 3. Вихід із системи та завершення роботи

У графічному інтерфейсі можна виконати:

- Зміну користувача через меню системи.
- Перезавантаження (Restart).
- Вимкнення (Power Off).

Для переходу в режим суперкористувача використовується команда:

```bash
sudo -i
```

## Інтерфейс мобільної ОС (iOS)

iOS використовує повністю сенсорний графічний інтерфейс. Головний екран містить іконки програм, папки та віджети. У верхній частині розташована панель стану.

Меню «Налаштування» дозволяє керувати Wi-Fi, Bluetooth, мобільними даними, звуком, екраном, конфіденційністю та оновленням системи.

Вхід у систему здійснюється через Face ID, Touch ID або пароль. Завершення роботи — через меню живлення або комбінацію кнопок.

# Контрольні запитання

### Наведіть приклади серверних додатків Linux

MySQL, PostgreSQL — для баз даних.  
Postfix, Sendmail — для розсилки.  
Samba, NFS, vsftpd — для файлообміну.

---

### Для чого потрібен менеджер пакетів?

Менеджер пакетів використовується для встановлення, оновлення та видалення програм з автоматичним керуванням залежностями.  
Приклади: apt, dnf, yum, pacman.

---

### Скільки віртуальних консолей у Linux?

Зазвичай 6 текстових консолей.  
Перемикання: Ctrl + Alt + F1–F6.
