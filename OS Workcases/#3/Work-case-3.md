# Work-case 4

# 1. Поняття «пакет» та «репозиторій». Огляд менеджерів пакетів

# Пакет
# Пакет — це архівований набір файлів, який містить програму або бібліотеку
# разом із усіма необхідними даними для її встановлення: виконувані файли,
# конфігурації, залежності та метадані (версія, опис, автор).
# Найпоширеніші формати пакетів: .deb (Debian/Ubuntu), .rpm (Fedora/RHEL).

# Репозиторій
# Репозиторій — це централізоване сховище пакетів, з якого система отримує
# програми для встановлення та оновлення. Репозиторії можуть бути офіційними,
# сторонніми або локальними.

# Огляд менеджерів пакетів

# APT
# Використовується в Debian/Ubuntu, працює з .deb пакетами.
# Автоматично встановлює залежності, простий у використанні.

# DNF
# Використовується в Fedora/RHEL, працює з .rpm пакетами.
# Має покращену продуктивність і підтримує історію операцій.

# YUM
# Старіший менеджер пакетів для RPM-систем, зараз замінений DNF.

# Pacman
# Використовується в Arch Linux. Швидкий, простий і мінімалістичний.

# Zypper
# Використовується в openSUSE. Потужний інструмент для роботи з пакетами.

# -------------------------------------------------------------------

# 2. Менеджер пакетів у моєму дистрибутиві (Ubuntu — APT)

# Оновлення списку пакетів
sudo apt update

# Оновлення системи
sudo apt upgrade

# Пошук пакета
apt search package_name

# Перегляд інформації про пакет
apt show package_name

# Список встановлених пакетів
apt list --installed

# Встановлення пакета
sudo apt install package_name

# Видалення пакета
sudo apt remove package_name

# Повне видалення (разом із конфігами)
sudo apt purge package_name

# Видалення непотрібних залежностей
sudo apt autoremove

# Додавання нового репозиторію
sudo add-apt-repository ppa:repo/name
sudo apt update

# -------------------------------------------------------------------

# 3. Встановлення програм через термінал

# Встановлення медіаплеєра (VLC)
sudo apt install vlc

# Встановлення середовища програмування (Python)
sudo apt install python3 python3-pip

# Або Java
sudo apt install openjdk-17-jdk
