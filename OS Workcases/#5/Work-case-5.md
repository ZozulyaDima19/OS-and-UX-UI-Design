# Work-case 5

## 1.
При роботі з персональним комп’ютером часто виникає потреба підключення периферійних пристроїв, таких як принтер або флеш-накопичувач. В операційній системі Linux для роботи з такими пристроями використовується механізм драйверів, файлової системи та служби автоматичного визначення пристроїв. Наприклад, при підключенні флешки система визначає її як окремий блочний пристрій (наприклад, /dev/sdb1), після чого виконується її монтування. Монтування — це процес підключення файлової системи пристрою до єдиного дерева каталогів Linux, що дозволяє отримати доступ до файлів. Зазвичай флешка автоматично монтується у каталог /media або /mnt, але це також можна зробити вручну за допомогою команди mount. Принтери у Linux працюють через систему CUPS, яка відповідає за керування друком, встановлення драйверів і обробку черги друку.

Суть операції монтування полягає у тому, що фізичний пристрій або розділ диска “підключається” до певної точки у файловій системі, після чого користувач може працювати з файлами так, ніби вони знаходяться у звичайній папці. Без монтування доступ до даних на пристрої неможливий. Для цього використовується команда mount, де вказується пристрій і точка монтування, а для відключення — команда umount.

Основна різниця між роботою з периферією у Linux та Windows полягає у підході до організації файлової системи та автоматизації. У Windows пристрої автоматично отримують букву диска (наприклад, D:, E:) і користувач працює з ними без додаткових дій. У Linux ж всі пристрої інтегруються у єдину файлову систему через монтування, що дає більше контролю, але потребує розуміння принципів роботи. Також у Linux частіше використовується термінал для керування пристроями, тоді як у Windows більшість операцій виконується через графічний інтерфейс.

## 2.
Для виконання практичної частини необхідно підключити флешку до віртуальної машини через налаштування гіпервізора, наприклад у VirtualBox через Devices → USB. Після підключення флешка автоматично з’явиться у файловому менеджері Linux, зазвичай у розділі “Інші розташування” або у каталозі /media. Через графічний інтерфейс достатньо відкрити флешку, вибрати файл і скопіювати його на робочий стіл або у домашню папку.

Для роботи через термінал спочатку потрібно перевірити підключені пристрої за допомогою команди lsblk або fdisk -l, знайти флешку (наприклад, /dev/sdb1), створити точку монтування та змонтувати її. Після цього можна скопіювати файл командою cp, наприклад з каталогу флешки у домашню директорію. Для відключення флешки використовується команда umount.

Щодо принтера, у графічному інтерфейсі Linux його можна додати через налаштування системи у розділі “Принтери”, де система автоматично знаходить пристрій або дозволяє додати його вручну. Після цього файл можна відкрити та надрукувати через стандартне меню друку. У терміналі друк можна виконати за допомогою команди lp, вказавши файл для друку.
